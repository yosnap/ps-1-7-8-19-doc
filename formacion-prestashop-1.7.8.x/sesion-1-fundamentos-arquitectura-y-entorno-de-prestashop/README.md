# Sesión 1: Fundamentos - Arquitectura y Entorno de PrestaShop

**Objetivo General de la Sesión:**&#x20;

Al finalizar esta sesión, habrás verificado que tu entorno de desarrollo PrestaShop con Docker está operativo. Comprenderás la arquitectura fundamental de PrestaShop 1.7 y sabrás cómo utilizar las herramientas básicas de depuración, incluyendo Xdebug con VS Code, sobre tu entorno ya configurado.
